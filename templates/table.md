# Table Template - Agent Instruction

1. export_csv - Export the table to a CSV file
2. analyze_data - Perform analysis: sums, averages, min/max, trends
3. filter_rows - Show only rows matching certain criteria
4. chart - Create a bar/pie/line chart from the data
